----- GameHandler -----
---MODEL---
  Handle 'JOIN' and 'CREATE' features.

  Attributes :
  . A list of GameLaunchers.

  Methodes :
  . Create : Add a Game Launcher to the list.
  . Join : join one of the GameLaunchers in the list.

---VIEW---
Handle the strides between the different game images.


----- GameLauncher -----
---MODEL---
  Handle the launching process.

  Attributes :
  . A list of the web sockets waiting for enough players .
  . Number of necessary players.
  . Max Number of players.
  . Current number of players.

  Methodes :
  .AddPlayer : add a socket to the list of sockets, parametrize the socket for polling and increments current Number
  of players, if current number of players equals max number of players, launch
  the game.
  .Launch : create an instance of the game with everything it needs to sustain itself
  then destroys the GameLauncher ?
  .Poll : if current number of players is between necessary and max, launch a poll
  that sends a yes/no image to all players. If the yes wins(without needing the no to win against the no)
  then launch.

---VIEW---
Handle the size of the game images in the saloon
